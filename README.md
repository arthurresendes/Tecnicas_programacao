## Técnicas de Programação 1: 

| Tópico                            | Conteúdo Principal                                                                 |
|------------------------------------|-----------------------------------------------------------------------------------|

| **Projetos/Desafios**              | Pequenos projetos e exercícios complexos para aplicação dos conhecimentos          |
